包的管理工具

npm、cnpm、yarn 

npm 

  npm config get registry  查看npm镜像   

  npm config set registry https://registry.npm.taobao.org  设置修改为淘宝镜像  

  npm install 包名 -g | -save -S | --save-dev -D 

  npm uninstall 包名 -g | -save -S | --save-dev -D 

  npm i 根据package.json 重新状态  

cnpm  

  npm install -g cnpm --registry=https://registry.npm.taobao.org
  
  cnpm install 包名 -g | -save -S | --save-dev -D 

  cnpm uninstall 包名 -g | -save -S | --save-dev -D 

  cnpm i 根据package.json 重新状态  

yarn 

  npm i yarn -g 安装yarn  

  yarn --version 查看yarn版本  

  yarn add 包名 -g | -save -S | --save-dev -D 

  yarn remove 包名 -g | -save -S | --save-dev -D 

  yarn 根据package.json 重新状态
