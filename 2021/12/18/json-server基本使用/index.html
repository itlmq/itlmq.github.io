hosjson-server

  1. 安装json-server 
npm i json-server -g

  1. 创建json文件设置接口的数据

  2. 使用命令运行接口 

json-server data.json   

--watch 实时监听

--host 设置ip

--post 端口号


json-server  --watch --host 10.41.153.11 --port 8888 data.json

常用的接口

请求所有的资源 

get http://127.0.0.1:3000/users

根据id请求资源 

get http://127.0.0.1:3000/users/:id

分页查找

get http://127.0.0.1:3000/users?_page=1&_limit=2

模糊查询 

 get http://127.0.0.1:3000/users?q=ls

添加资源 

post http://127.0.0.1:3000/users
